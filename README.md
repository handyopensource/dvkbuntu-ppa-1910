# adresse du ppa :  
  
**ppa:dvkbuntu/dvkbuntu-ppa**
  
## pour ajouter le ppa sur kubuntu 19.10 exécuter les commandes suivantes :  
  
```
sudo add-apt-repository ppa:dvkbuntu/dvkbuntu-ppa-19.10
sudo apt update
sudo apt install dvkbuntu
```
